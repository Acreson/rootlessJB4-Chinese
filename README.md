# #使用笔记



二进制文件位于:/var/containers/Bundle/iosbinpack64

越狱结构可以在/var/containers/ bundle /tweaksuppoet/找到

-符号链接包括:/var/LIB， /var/ulb， /var/bin， /var/sbin， /var/Apps， /var/libexec

根文件系统将保持只读，只要不乱/var，你将是安全的。
